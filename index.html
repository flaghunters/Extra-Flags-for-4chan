Extra flags for 4chan

Originally made for /int/, but we support /sp/, /pol/, and /bant/ now as well. Made by /ref/ and flags contributed by various /ref/ and /int/ posters. Your flag not here? Create an issue on the gitlab page

HOW TO INSTALL (Desktop)

  1. Install a userscript engine extension:
    • For Chrome and most other browsers*, install Tampermonkey from the Chrome Store
    • For Firefox, install Violentmonkey or Tampermonkey from the Firefox add-ons store
    • For Opera, install Tampermonkey from the Opera addons store
    • For Safari, install Tampermonkey from the Mac App Store
    • Greasemonkey was an option years ago. However, they decided to drop compatibility with older APIs making it useless
    • *most browsers that aren't Firefox or Opera, like Edge, Brave, Yandex, etc... are based on Chromium so they work with Chrome extensions just fine. Duckduckgo currently does NOT support extensions at all
  2. Install the Extraflags script by clicking here: Extra Flags for int.user.js
  3. Browse to /flag/
  4. A popup should appear asking to set your regions. Click "OK" and set your regions
  5. If it doesn't work, try to enable "allow user scripts" in your browser's settings. If it still doesn't work (unlikely) try to turn on "Developer Mode" in your browser
  6. To change regions, click the extension menu>Extra Flags setup
  7. Become a regular on /flag/ :^)

HOW TO INSTALL (Mobile)

Extraflags works on mobile with browsers that support extensions, like Yandex Browser or KiwiBrowser

Disclaimers for phone users:

  • Extraflags doesn't work on Clover, Omnichan, Kuroba and similar apps. Those are gay anyway so who cares
  • Chrome for Android doesn't support extensions (and has never supported extensions)
  • Extraflags (as everything else) can't, doesn't work on- and likely never will be supported by iPhones
  • Firefox and Opera mobile browsers now support extensions, after a long period when they didn't

    • Instructions:

    1. Install Yandex mobile (or any other suitable mobile browser) from the Google Play Store
    2. Once inside the mobile browser, install Tampermonkey from the Chrome Store (you don't have to install any Tampermonkey app from the Play Store or anything like that) or from their stores if you are on Firefox or Opera
    3. Install the script: Extra Flags for 4chan-Mobile.user.js
    4. Browse to /flag/
    5. A popup should appear asking to set your regions. Click "OK" and set your regions
    6. If it doesn't work, try to enable "allow user scripts" in your browser's settings. If it still doesn't work (unlikely) try to turn on "Developer Mode" in your browser
    7. To change regions, click the extension menu>Extra Flags setup
    8. Become a regular on /flag/ :^)
