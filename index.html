Extra flags for 4chan

Originally made for /int/, but we support /sp/, /pol/, and /bant/ now as well. Made by /ref/ and flags contributed by various /ref/ and /int/ posters. Your flag not here? Create an issue on the gitlab page

HOW TO INSTALL (Desktop - Firefox/Chrome/Opera)

Instructions:

  1. Download Violentmonkey or Tampermonkey for Firefox, Tampermonkey for Chrome, or Violentmonkey for Opera.
    Greasemonkey was an option years ago. However, they decided to drop compatibility with older APIs making it useless.
  2. Install the Extra Flags script: Extra Flags for int.user.js.
  3. Browse to /int/
  4. If you're using Tampermonkey, a page asking for permissions may appear. Click "Always allow domain"
  5. Click "OK" to set your region
  6. Click "Always allow domain," a second time
  7. Set your region
  8. Reload the page
  9. Post from Quick Reply box (tap/click on a post number; top form does not work)
  10. To change regions, click the Violentmonkey/Tampermonkey menu->Extra Flags setup, and return back to the 4chan page to set your region from the popup menu

HOW TO INSTALL (Mobile)

Extra Flags works on mobile with browsers that support extensions, like Yandex Browser or KiwiBrowser.

Disclaimers for phone users:

  • Extra Flags doesn't work on Clover, Omnichan, Kuroba and similar apps.
  • Chrome for Android doesn't support extensions (and has never supported extensions).
  • Extra Flags (as everything else) can't, doesn't work on- and will never be supported by iPhones.
  • Firefox for Android supports some useful extensions again after a long period of mental retardation.

    • Instructions for Mobile Users:

      If you choose Firefox for Android:
    1. Install Firefox for Android from the Google Play Store
    2. Install ViolentMonkey or Tampermonkey from Mozilla's Add-Ons page
      If you choose Yandex Mobile:
    1. Install Yandex mobile from the Google Play Store
    2. Install ViolentMonkey or Tampermonkey from the Chrome Store
      Then:
    1. Install the Mobile script Extra Flags for 4chan-Mobile.user.js
    2. Browse to /int/
    3. A page asking for permissions may appear. Click "Always allow domain"
    4. Click "OK" to set your region
    5. Click "Always allow domain," a second time
    6. Set your region
    7. Reload the page
    8. To change regions, open the Extensions menu, click the Violentmonkey/Tampermonkey menu->Extra Flags setup, and set your region from the popup menu
