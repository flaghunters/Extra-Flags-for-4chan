Extra flags for 4chan

Originally made for /int/, but we support /sp/, /pol/, and /bant/ now as well. Made by /ref/ and flags contributed by various /ref/ and /int/ posters. Your flag not here? Create an issue on the gitlab page

HOW TO INSTALL

Prerequisites:

Be sure to have
  1. A browser that supports extensions
  2. A user script engine extension for that browser

For Firefox for Desktop and Firefox for Android: you can download Violentmonkey or Tampermonkey
For Chrome Desktop, Opera and similar: consider Violentmonkey or Tampermonkey
On mobile: you can consider Yandex Browser or KiwiBrowser, for which the Chrome extensions can be installed.

Disclaimers for phone users:

  • Extra Flags doesn't work on Clover, Omnichan, Kuroba and similar apps.
  • Chrome for Android doesn't support extensions (and has never supported extensions).
  • Extra Flags (as everything else) can't, doesn't work on- and likely never will be supported by iPhones.

    • Instructions:

    1. Open your browser that supports user script engine extensions and navigate to this page.
    2. Install the Extra Flags script by clicking the link: Extra Flags for int.user.js.
    3. Browse to /int/
    4. If you're using Tampermonkey, a page asking for permissions may appear a few times. Click "Always allow domain" to allow access to the server that stores flags and to gitlab to download flag images.
    5. On first use a pop-up should appear to asking to set your region. Click "OK".
    6. Set your region by using the available buttons and click "Save regions."
    7. You can now post from Quick Reply box (tap/click on a post number; top form does not work).
    8. To change regions, click the Violentmonkey/Tampermonkey menu->Extra Flags setup, and return back to the 4chan page to set your region from the popup menu.
